Angol-magyar nyelvészeti szakszótár v.2.0.0 (2022-10-01)

Verziótörténet (Changelog):
Webfejlesztés, design: Dr. Porkoláb Ádám
Az adatbázisban található szócikkek száma a 2.0.0-ás verzióban: 12384
  • 2.0.0 (2022-10-01)
    • - A Heroku deploy megszűnése miatt a NyelvSzó saját tárhelyre, elvileg a végleges helyére költözött.
    - A Spring Boot-os applikáció teljes újraírásra került Angular 14-ben, TypeScript alapon. A backendet egy saját fejlesztésű Express-es alkalmazás adja.
    - NyelvSzó logó megtervezése.
    - Weblapdesing fejlesztése, frontend fejlesztése.
    - JWTWebtoken bevezetése és az adminfelület, 'sózott' (hashed) jelszóval védése megtörtént.
    - A kezelőfelület és az alapvető szótári kifejezések többnyelvűsítését implementáltam.
  • 1.3 (2021-10-21)
    • - Teljes REST API támogatás elkészült.
    - Adminfelület kiépítése és a CRUD-műveletek frontendes megvalósítása.
    - Az adminfelület további fejlesztése megtörtént: felhasználók kezelésének kiépítését megvalósítottam.
    - Weblapdesing fejlesztése, frontend fejlesztése.
    - Spring Security bevezetése és az adminfelület jelszóval védése megtörtént.
  • 1.0 (2021-10-08)
    • - Az átnézett és részben javított szótári anyagot - melyet a kiadványban szereplő változat alapján fejlesztettünk tovább - JSON formátumba mentettük és H2 adatbázis készült belőle.
    - Az alapvető keresőfelület elkészült. Felhasznált technológiák: Bootstrap (HTML és CSS3), Spring Boot (Thymeleaf, Lombok)
    - A www.nyelvszo.eu domain megvásárlása.

  • Tervek a későbbi verziókra:
    • - További desingbeli fejlesztés.
    - A szótári anyag fejlesztése és hibajavítása.
