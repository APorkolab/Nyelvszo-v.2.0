Angol-magyar nyelvészeti szakszótár v.2.1.0 (2024-08-22)

Verziótörténet (Changelog):
Webfejlesztés, design: Dr. Porkoláb Ádám
Az adatbázisban található szócikkek száma a 2.1.0-ás verzióban: 12.385 db
  • 2.1.0 (2024-08-22)
    • Dokku deploy támogatás: Az alkalmazás felkészítése a Dokku-ra történő telepítésre, ezzel biztosítva a könnyebb skálázhatóságot és üzemeltetést.
    • SQL adatbázistárolás támogatás: Az alkalmazás átalakítása SQL alapú adatbázis kezelésére, amely nagyobb teljesítményt és jobb adatkezelési képességeket biztosít.
    • `AuthGuardService` és `RoleGuardService` fejlesztése: Aszinkron adatkezelés támogatása a felhasználói szerepkörök valós időben történő ellenőrzésére, valamint a nyilvános oldalak kezelése, amelyek lehetővé teszik a hozzáférést bejelentkezés nélkül is.
    • `JwtInterceptor` továbbfejlesztése: A token lejárati idejének kezelése és automatikus token frissítés bevezetése a felhasználói élmény javítása és a biztonság növelése érdekében.
    • `NotificationService` kiterjesztése: Alapértelmezett címek és egységes értesítési kezelés bevezetése a `ngx-toastr` használatával, hatékonyabb értesítési mechanizmusokat biztosítva a felhasználók számára.
    • Általános hibajavítások és refaktorálás: A kód tisztítása, felesleges ismétlések eltávolítása, valamint a modern Angular és TypeScript ajánlásoknak megfelelő fejlesztések elvégzése.
    • `throwError` új szintaxisának alkalmazása: A legújabb RxJS verzió támogatásának bevezetése a hibakezelés megfelelő működésének biztosítása érdekében.
    • Swagger dokumentáció: Teljes értékű Swagger dokumentáció készült a backendre vonatkozóan, amely elérhető a következő címen: https://www.api.nyelvszo.eu/api-docs/.
    • Fejlesztési dokumentáció frissítése: Az új fejlesztések, deploy lépések és adatbázis-kezelés dokumentálása a könnyebb karbantarthatóság érdekében.
  • 2.0.0 (2022-10-01)
    • Heroku migráció: Az ingyenes Heroku deploy megszűnése miatt a NyelvSzó saját tárhelyre, elvileg a végleges helyére költözött.
    • Teljes frontend újraírás: A Spring Boot alapú alkalmazás teljes újraírásra került Angular 14-ben, TypeScript alapon. A backendet egy saját fejlesztésű Express-es alkalmazás biztosítja.
    • Logó és design: NyelvSzó logó megtervezése és weblapdesign fejlesztése.
    • JWT Web Token bevezetése: Az adminfelület biztonságának növelése `sózott` (hashed) jelszavakkal és JWT token használatával.
    • Többnyelvűsítés: A kezelőfelület és az alapvető szótári kifejezések többnyelvűsítésének implementálása.
  • 1.3 (2021-10-21)
    • REST API támogatás: Teljes REST API támogatás elkészült.
    • Adminfelület kiépítése: CRUD-műveletek frontendes megvalósítása, felhasználók kezelésének kiépítése.
    • Frontend fejlesztések: Weblapdesign fejlesztése és a felhasználói élmény optimalizálása.
    • Spring Security bevezetése: Az adminfelület jelszóval védése és biztonsági intézkedések bevezetése.
  • 1.0 (2021-10-08)
    • Szótári anyag feldolgozása: Az átnézett és részben javított szótári anyagot JSON formátumba mentettük és H2 adatbázis készült belőle.
    • Keresőfelület kialakítása: Az alapvető keresőfelület elkészült. Felhasznált technológiák: Bootstrap (HTML és CSS3), Spring Boot (Thymeleaf, Lombok).
    • Domain megvásárlása: A www.nyelvszo.eu domain megvásárlása.

  • Tervek a későbbi verziókra:
    • További designbeli fejlesztés.
    • A szótári anyag fejlesztése és hibajavítása.
    • API dokumentáció bővítése és a szótárhoz kapcsolódó szolgáltatások kibővítése.
